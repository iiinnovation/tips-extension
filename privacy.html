Data Collection

Tips does not collect any personal data. Specifically:

  • We do not collect your browsing history
  • We do not collect the content you save
  • We do not collect usage analytics
  • We do not use cookies or tracking pixels
  • We do not require account registration

Data Storage

All data created by Tips is stored locally using Chrome's built-in storage API (chrome.storage.local). This includes:

  • Saved content clips
  • Tags and categories
  • User preferences and settings

This data remains on your device and is never transmitted to any external server.

Permissions Explained

Tips requests the following browser permissions:

  • storage: To save your clipped content locally
  • unlimitedStorage: To allow saving more content without hitting browser limits
  • activeTab: To access the current page when you trigger the save shortcut
  • clipboardWrite: To enable the "Copy" feature in the popup
  • alarms: To schedule automatic cleanup of old content

Third-Party Services

Tips does not integrate with any third-party services, analytics platforms, or advertising networks.

Data Export

You can export all your saved content at any time using the Export feature. The exported file is a standard Markdown document that you control entirely.

Data Deletion

You can delete individual clips or clear all data at any time through the Settings panel. Uninstalling the extension will also remove all stored data.
